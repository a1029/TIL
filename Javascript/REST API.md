# REST
- HTTP를 기반으로 클라이언트가 서버의 리소스에 접근하는 방식을 규정한 아키텍처
- REST API : REST를 기반으로 서비스 API를 구현한 것
- RESTful : REST의 기본 원칙을 성실히 지킨 서비스 디자인을 RESTful 이라고 표현

## REST API 설계 원칙
- URI는 리소스를 표현
- 리소스에 대한 행위는 HTTP 요청메서드로 표현
